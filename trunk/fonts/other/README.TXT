============================
	TeX Pi fonts
============================

Now in fully hinted ATM compatible Adobe Type 1 format 13 `pi' fonts:


BBOLD (Alan Jeffrey Geometric Sans Serif Blackboard Bold)

			bbold5, bbold7, bbold10

RSFS (Ralph A. Smith Formal Script face based on R. Hunter Middeleton)

			rsfs5, rsfs7, rsfs10

STMARY (Jeremy Gibbons and Alan Jeffrey St. Mary's Road Symbolic Logic)

			stmary5, stmary7, stmary10

WASY2 (Roland Waldi extension of LASY symbol - version 2.0)

			wasy5, wasy7, wasy10, wasyb10
