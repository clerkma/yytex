You can use the EM fonts with an existing plain TeX format.
Just add \input emplain to your source file.

In this directory you will also find plain-em.tex used
for making a new plain TeX format which refers to the EM font set.

Run ini-TeX on plain-em.tex to create a plain-em.fmt file.
